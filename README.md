# Markdown Links
![](https://github.com/ljohanaalvarez/bog001-md-links/blob/alternativa/images/Md-Links%20(1).png)

## Instalación

* Ejecute el comando

  npm install @ljohanaalvarez/md-links

## Usando mdlinks

* Si ejecuta:

  npx mdlinks <path-to-file>

  Por ejemplo:
  
    npx mdlinks ./proofReadme.md
